# EU4 and Stellaris WinXP fix
Patch for EU4, HoI 4 and Stellaris to work on Windows XP.

## Supported games:
- Europa Universalis 4: 1.6.1 - 1.6.3
- Hearts of Iron 4: 1.0.0 - 1.0.1-beta2
- Stellaris 1.0.0 - 1.1
